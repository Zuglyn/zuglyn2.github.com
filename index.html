Selectores y Reglas Visuales

Selectores css y Etiquetas

Etiqueta Nombre descripcion
#ID Selector de ID Los selectores de ID se indican con un signo de almohadilla # seguido del nombre de ID.
<link> Link etiqueta El código CSS se puede escribir en sus propios archivos para mantenerlo separado del código HTML.
h1, h2 Grupos de selectores CSS Haga coincidir varios selectores con la misma regla CSS, utilizando una lista separada por comas.
<style> Escribir CSS en un archivo HTML El código CSS se puede escribir en un archivo HTML al incluir el código en <style> etiquetas.
!important !important Regla La !important regla CSS se utiliza en declaraciones para anular cualquier otra declaración de una propiedad e ignorar la especiİcidad del selector.

Propiedades CSS

Propiedad Nombre descripcion
background-image background-image establece la imagen de fondo de un elemento. Se debe proporcionar una URL de imagen en la sintaxis url("moon.jpg") como valor de la propiedad.
url() URL funcion En CSS, la url() función se utiliza para envolver URL de recursos. Estos se pueden aplicar a varias propiedades, como background-image
background-color Color de Fondo La background-color propiedad CSS controla el color de fondo de los elementos.
text-align Alinear Texto La text-align propiedad CSS se puede utilizar para establecer la alineación del texto de los contenidos en línea. Esta propiedad se puede establecer a estos valores: left , right o center
color color del texto de primer plano Usando la color propiedad, el color del texto de primer plano de un elemento se puede establecer en CSS. El valor puede ser un nombre de color válido compatible con CSS como green o blue . Además, se puede utilizar un código de color de 3 o 6 dígitos como #22f o #2a2aff para establecer el color.
